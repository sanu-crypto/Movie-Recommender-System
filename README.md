# 🎬 Movie Recommender System

A Machine Learning based Movie Recommendation Web App built using **Streamlit**, which suggests movies based on user selection. It also displays posters, trailers, ratings, and reviews using the TMDB API.

## 📌 Features

- 🎯 Content-based Movie Recommendation
- 🎬 Displays Movie Posters
- ▶️ Watch Movie Trailers
- ⭐ Shows Ratings
- 📝 Displays Reviews
- 🎲 "Surprise Me" Feature
- 🔥 Trending Movies Section
- 🎨 Attractive UI (Netflix-style)

---

## 🧠 How It Works

- Uses **TF-IDF Vectorization** on movie descriptions
- Computes similarity using **Cosine Similarity**
- Recommends top 5 similar movies based on selected movie
